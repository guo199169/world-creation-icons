# 世界创建图标库 🌍

一个精心设计的图标集合，主题为"世界创建"（World Creation）。

## 📦 包含内容

- `icons/` - 所有图标文件（SVG 和 PNG 格式）
- `svg/` - 原始 SVG 矢量图标
- `png/` - PNG 位图图标（多种尺寸）
- `docs/` - 文档和使用指南
- `examples/` - 使用示例

## 🚀 快速开始

### 使用 SVG 图标

```html
<svg width="24" height="24" xmlns="http://www.w3.org/2000/svg">
  <use href="./svg/icon-name.svg#icon-name"></use>
</svg>
```

### 使用 PNG 图标

```html
<img src="./png/icon-name-24.png" alt="图标名称" width="24" height="24">
```

## 📁 目录结构

```
world-creation-icons/
├── icons/          # 所有图标（按分类组织）
├── svg/            # SVG 源文件
├── png/            # PNG 文件（16x16, 24x24, 48x48, 64x64）
├── docs/           # 文档
│   └── usage.md    # 使用指南
├── examples/       # 示例代码
└── README.md       # 本文件
```
